ssdlut-news-hub
===============

大连理工大学软件学院新闻聚合站点

ubuntu下使用方法,其他系统类似：
[1]安装必要的python第三方包
$ sudo easy_install requests
$ sudo easy_install beautifulsoup
$ sudo easy_install redis
$ sudo easy_install apscheduler
[2]安装redis-server
$ sudo apt-get install redis-server
[3]运行info.py
